# whnj5shdxb-sudo

A monorepo project.

## Project Structure

```
.
├── packages/           # Workspace packages
│   └── core/          # Core package
├── package.json       # Root package.json with workspaces
└── README.md          # This file
```

## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm >= 9.0.0

### Installation

```bash
npm install
```

### Scripts

- `npm run build` - Build all packages
- `npm run test` - Run tests in all packages
- `npm run lint` - Lint all packages
